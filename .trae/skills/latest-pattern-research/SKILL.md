# Latest Pattern Research

这个 Skill 用于承载“联网看最新范式，再转成当前系统可落地建议”的固定动作。

它不是泛研究 agent，而是自我优化 loop 中的一个研究节点，目标是避免系统只围绕旧经验修补，却错过更好的近期范式。

## 适用场景

- 宿主明确要求参考最新热点范式
- 当前问题可能有更好的新解法
- 进入自我优化 loop 的方案设计阶段

## 输入

- 当前问题定义
- 宿主重点诉求
- 当前系统结构和限制
- 需要关注的技术领域或产品范式

## 输出

- 最新范式摘要
- 可借鉴模式
- 不建议采用的模式
- 采用收益与风险
- 对当前系统的落地建议

## 规则

- 必须优先看官方文档、主流框架说明或高可信实践
- 不允许编造“热点”与趋势
- 必须把“适合当前系统”与“只适合别的系统”区分开
- 输出要能直接服务于后续改造决策
