# Catalog Import Check

这个 Skill 用于承载原来 `catalog-quality-agent` 更偏检查型的职责。

当货盘输入的规则已经比较稳定时，导入质量检查更适合做成 skill，而不是继续作为一个需要开放式推理的 agent。

## 适用场景

- Excel / 表格导入后做初始体检
- 检查字段映射是否完整
- 检查缺失值、重复值、异常值
- 输出风险等级与是否允许继续推进

## 输入

- 导入结果
- 表头映射
- 批次信息
- 预期字段清单

## 输出

- 质量摘要
- 问题清单
- 风险等级
- 是否允许继续推进

## 规则

- 优先输出结构化问题，而不是模糊判断
- 不负责候选品策略判断
- 若发现严重问题，应阻断进入后续电商节点
