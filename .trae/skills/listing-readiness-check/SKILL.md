# Listing Readiness Check

这个 Skill 用于承载原来 `listing-readiness-agent` 更偏门槛检查的职责。

如果“是否可上架”的规则主要来自字段完整性、素材齐备度和平台准备度，那么这一步更适合 skill，而不是独立 agent。

## 适用场景

- 上架前检查资料是否齐全
- 检查主图、详情、标题、价格、类目、供应信息是否完整
- 输出缺失项与风险摘要

## 输入

- 商品结果
- 供应商结果
- 素材结果
- 平台检查项

## 输出

- 上架准备结论
- 缺失项
- 风险摘要
- 是否建议进入人工确认

## 规则

- 不直接发布商品
- 不替代创意与供应商判断
- 重点是门槛检查与结构化输出
